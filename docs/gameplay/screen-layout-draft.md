# 全体的な事

* 牌は今文字として表現されているが、最終的には画像で表示される予定
* 将来的に牌は基本的にはクリック又はドラッグができるようにする

# レイアウト

* 画面の左側を「卓上」及び「基本動作」、右側を情報エリアとする

## 卓上

* 牌山は四角形に並んでいる
  * 個数はできるだけ均等になるのが望ましいが、必須ではない
  * 牌山は2段に積まれているが、上の牌が見えればいい
  * (のちの実装では、牌山をクリックすると上の牌がめくれるようにする。さらにその牌をクリックするとその牌を内側によけ、下の牌がめくれるようにする)
* 手牌は牌山の外側にあり、牌山に対して平行に並んでいる
  * プレイヤーは「回り」が決まっていれば相対的な位置を保っていればよく、自分以外のプレイヤーは以下のように見える
    * 自分は手前側(下側)にいる
    * 4人である場合は全ての辺が埋まるので考えなくていい
    * 3人である場合は、自分の向い側が空席になり、両側に他プレイヤーがいるようにする
    * 2人である場合は、相手は自分の向い側にいるようにする
* 捨牌は各プレイヤーの前、牌山の更に中央よりに置かれる
  * 捨牌はプレイヤーの目の前の牌山に対して平行に並んでいる
    * 両隣のプレイヤーの捨牌は、自分から見て縦に並ぶ事になる
  * 各々のプレイヤーから見て左側から並び、新しい牌が右側に置かれる
  * 1列には6つの捨牌が並び、7つ目以降、6個毎に新しい列ができる
* 卓の中央に「卓の状態」「誰か1人についている状態」が表示される

## 基本動作

* ゲーム進行に対するボタン
  * 「ツモ」(ショートカットキー [Space] でも可 )
    * 山の先頭から牌を一つ手牌にもってくる
  * 発声
    * 定義された発声がボタンになっている。ボタンを押すとその発声をした事が他プレイヤーに伝えられる
  * 手牌を開ける
    * 手牌を全て他のプレイヤーに公開する
* 操作モードの切り替え
  * 捨てる/開ける モードの切り替え
    * 「捨てる」モード: 手牌をクリックすると、その手牌が河に捨てられる
    * 「開ける」モード: 手牌をクリックすると、その牌が全体に公開される
    * これらの「クリック」はドラッグ動作と両立したい

## 情報

* 表示されているもの
  * 卓の状態
    * 全てのプレイヤーが操作でき、共有される
  * プレイヤー一覧
    * 名前
    * 点数
    * プレイヤーに付与されている状態
* 操作できるもの
  * 自分の状態
    * 個々に付与されている状態ならそれを切り替える事ができる
    * 卓内に1人が付与される状態であればそれを自分に付与することができる
  * 点数の譲渡
    * 譲渡先、譲渡元が自分であるような点数の操作ができる
